Приложение для редактирования Master-Detail таблиц
Master содержит список документов, Detail - список спецификаций по каждому документу
Инструкция по сборке и запуску проекта MDRT

1.	Основная информация
Используемая СУБД: PostgreSQL 14
• Язык: Java 17
• Для разработки проекта использовались Spring framework и Maven
• Проект загружен в репозиторий GitHub: https://github.com/Zephir-9/MDRT
• Используемая IDE: IntelliJ IDEA
• Приложение для работы с БД: DBeaver
2.	Создание базы данных
• Распаковать содержимое архива или скопировать проект с репозитория GitHub
• Запустить сервер PostgreSQL
• Запустить в PostgreSQL скрипт dump-MDRT-202511242012.sql из папки проекта для создания БД MDRT
3.	Сборка
• Открыть в командной строке путь к проекту MDRT
• Выполнить сборку проекта: mvn clean package (если установлен maven).
Или открыть проект в IDEA и собрать проект в нём, установив плагин maven
• Запустить проект с профилем local: java -jar target\MDRT-1.0-SNAPSHOT.jar --spring.profiles.active=local
• Перейти в браузере по адресу: http://localhost:9191/mdrt-test/api/
